# NestDeck

NestDeck is a Windows client application to control your pc directly from a Google Nest Hub

## Installation

Compile this project (C#) or download a compiled Setup from [nestdeck.base64.pt](https://nestdeck.base64.pt/)


## Usage

After having the client running, open the Deck Editor and login with your google account. 
From the web editor you'll be able to create and edit decks.

Options in the client:
- Start cast automatically - If you've connected to a Nest Hub before, the cast will automatically start to that device the next time this client is loaded
- Start when Windows starts - Auto run the client on windows boot
- Start in system tray - Auto minimize to tray
- Auto change deck - When ON, alerts the Nest Hub of the active window. It will look if there's a deck with matching process name and auto change decks.
